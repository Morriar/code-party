# How to feedback

Fork this repository!

```bash
git clone https://github.com/#{your_github_account}/code-party.git
cd code-party
git switch -c rubykaigi-2024-feedback
cp rubykaigi-2024/feedback.yaml rubykaigi-2024/feedback/#{your_github_account}.yaml
#{your_favorite_editor} rubykaigi-2024/feedback/#{your_github_account}.yaml
git add rubykaigi-2024/feedback/#{your_github_account}.yaml
git commit rubykaigi-2024/feedback/#{your_github_account}.yaml
git push origin rubykaigi-2024-feedback
```

Send a pull request!
